Spacing

Bootstrap includes a wide range of shorthand responsive margin and padding utility classes to modify an element’s appearance.Assign responsive-friendly margin or padding values to an element or a subset of its sides with shorthand classes. Includes support for individual properties, all properties, and vertical and horizontal properties. Classes are built from a default Sass map ranging from .25rem to 3rem.

More about this can be found here:

Layout/Containers

Components and options for laying out your Bootstrap project, including wrapping containers, a powerful grid system, a flexiple media object, and responsive utility classes. Containers are the most basic layout element in Bootstrap and are required when using our default grid system. Choose from a responsive, fixed-width, container (meaning its max-width changed at each breakpoint) or fluid width 9meaning it's 100% wide all the time). Can be nested, but most layouts do not require a nested container.


To control how many columns (bootstrap has a total of 12 columns on a page), simply write col-# of columns.

Mobile First

Bootstrap includes a powerful mobile-first flexbox grid to build layouts of all shapes and sizes thanks to a twelve column system, five default responsive tiers, Sass variables and mixins, and dozens of predefined classes.

Documentation and examples for badges, our small count and labeling component. Badges scale to match the size of the immediate parent element by using relative font sizing and em units. Badges can be used as part of links or buttons to provide a counter. Note that depending on how they are used, badges may be confusing for users of screen readers and similar assistive technologies. While the styling of badges provides a visual cue as to their purpose, these users will simply be presented with the content of the badge. Depending on the specific situation, these badges may seem like random additional words or numbers at the end of a sentence, link, or button. Unless the context is clear (as with the “Notifications” example, where it is understood that the “4” is the number of notifications), consider including additional context with a visually hidden piece of additional text.

My Opinion on Bootstrap

By linking a website to a Boostrap server, you are given access to their library of presets (which are customizable). It saves the time from manually implementing things into CSS, such as hover states. Using this is especially useful for government sites, as ones only needs basic knowledge of HTML & CSS to use it. I like it because it is convenient.
